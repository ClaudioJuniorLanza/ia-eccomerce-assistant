# IA E-commerce Assistant

Este é o repositório para o projeto de estudo de um assistente de IA integrado ao ciclo de desenvolvimento de software, utilizando uma plataforma de e-commerce simplificada como base.

O objetivo principal é explorar como uma IA pode aprender com o histórico do projeto, decisões de arquitetura, discussões e código para se tornar uma fonte de conhecimento ativa e consultável para a equipe de desenvolvimento.
